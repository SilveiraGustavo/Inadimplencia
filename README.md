# Requisitos para Execução do Projeto

Antes de iniciar, certifique-se de possuir o Python instalado em sua máquina.

## 1. Clonar o Repositório

Realize o clone do projeto utilizando o comando abaixo:

```bash
git clone https://github.com/SilveiraGustavo/Inadimplencia.git
```

Após o clone, acesse a pasta do projeto:

```bash
cd inadimplencia
```

## 2. Criar o Ambiente Virtual

Recomenda-se a criação de um ambiente virtual para isolar as dependências do projeto.

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Linux/macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

## 3. Instalar as Dependências

O projeto possui um arquivo `requirements.txt` contendo todas as bibliotecas utilizadas durante o desenvolvimento.

Execute o comando abaixo para instalar as dependências e recriar o ambiente do projeto:

```bash
pip install -r requirements.txt
```

Após a instalação, o ambiente estará configurado e pronto para execução do projeto.
